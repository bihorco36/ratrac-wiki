# Spielsystem

import Ultiplays from '../../components/ultiplays'

Wir spielen tendenziell im [Telemark](#telemark) System und haben als Fallback ein [Gerlach Erwin](#gerlach-erwin). In der Endzone entwickeln wir uns zu einer [Redzone](#redzone)

### Key Points

* Nie einen Dump auf dem Bruch
* Kein Fixes Handling
* Kommunikation für Angebote

## Telemark

Unser Telemark System ist ein Splitstack mit einem 1-2er Handling. Auch hier wollen wir auf gar keinen Fall einen Dump auf dem Bruch


## Gerlach Erwin

Das Gerlach Erwin ist ein Vertical Stack auf einer drittels Linie, abhängig davon, ob wir auf der Open Side sind oder nicht.
